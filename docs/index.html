About Windows Security Warning

Why does Windows show a warning?

When you download KeyBase, Windows may display a "Windows protected your PC" message with warnings like "This app might put your PC at risk." This happens because:

  • üîê KeyBase is not digitally signed ‚Äî Digital code-signing certificates cost hundreds of dollars annually. As an open-source project, we haven't purchased one yet.
  • üìä Limited download history ‚Äî Windows SmartScreen flags files that haven't been downloaded by many users yet.
  • ‚úÖ This is completely normal for new/small software projects ‚Äî It doesn't mean the software is unsafe.

‚úì KeyBase is Safe

KeyBase is a legitimate, open-source desktop application:

  • üìÇ Open source ‚Äî All code is publicly available on GitHub
  • üè† Runs locally ‚Äî No data is sent to external servers
  • üíæ Embedded database ‚Äî Your data stays on your computer
  • üîç Transparent ‚Äî You can review the source code yourself

If you trust this project and want to install KeyBase, simply follow the steps below to bypass the Windows warning.

Step-by-Step Installation Instructions

Step 1

Download KeyBase Setup

Click the download button above or visit the GitHub Releases page and download KeyBase-Setup.exe. Save the file to your computer.

Step 2

Handle Windows Security Warning

Double-click KeyBase-Setup.exe to start the installation. Windows will show a security warning: "Windows protected your PC".

Step 3

Bypass Windows SmartScreen (Click "More info")

When you see "Windows protected your PC", click "More info" to reveal additional options. Then click the "Run anyway" button that appears.

Step 4

User Account Control Confirmation

Windows User Account Control (UAC) will ask for permission. Click "Yes" to allow the installer to make changes to your device.

Step 5

Select Destination Location

The setup wizard will start. Choose where you want to install KeyBase or keep the default location (C:\Program Files\KeyBase). Click Next to continue.

Step 6

Select Start Menu Folder

Choose the Start Menu folder where shortcuts will be created, or use the default "KeyBase" folder. Click Next.

Step 7

Create Desktop Icon (Optional)

Choose whether to create a desktop shortcut for quick access. Check the box if you want a desktop icon, then click Next.

Step 8

Ready to Install

Review your installation settings. If everything looks good, click Install to begin the installation process.

Step 9

Installing KeyBase

The installer will copy files to your computer. This may take a minute or two. Wait for the progress bar to complete.

Step 10

Installation Complete!

Installation is finished! You can choose to launch KeyBase immediately by keeping the checkbox selected. Click Finish to complete the setup.

Step 11

Start Using KeyBase!

KeyBase is now installed and ready to use! Launch it from the Start Menu, desktop shortcut, or the launch checkbox. The main window will appear where you can start adding customer records with fast keyboard navigation.
